DevChecked.com

A Web Freelancers Speedy Companion

DevChecked is a project management application targeted at freelancers in the web industry. Some of the core features include:

  • Unique User Accounts
  • User Dashboard
  • Custom user project templates
  • Project changes auto-save to database.
  • One click client update email generation.
  • Searchable address book
  • File manager / Client download page generator.

Contributions

  • Documentation including the project proposal, weekly updates, change requests, hand-off packet, and server/app installation instructions.
  • Wireframing and Design Comps
  • Front-end development (HTML, CSS, JavaScript, jQuery)
  • Back-end development (PHP, MySql, database design/build)
